#### Задание 1: Модели и база данных
- Описать две модели SQLAlchemy: CarDealer и Car.  
Связать их отношением один-ко-многим.
Реализовать миграцию базы данных: таблицы должны создаваться автоматически.
Требования к моделям:

## CarDealer (Автосалон):

- id  
- name (уникальное название автосалона)  
- city  
- address  

## Car (Автомобиль):

- id  
- brand  
- model
- year
- price
- dealer_id (FK)
- created_at

#### Задание 2: Маршруты и шаблоны Flask
Реализовать маршруты:  

## Главная страница
### GET /  
- Список всех автосалонов.
- Для каждого автосалона отображается количество автомобилей в наличии.

## Страница автосалона
### GET /dealer/int:dealer_id
- Список автомобилей выбранного автосалона.

## Добавление автомобиля
### GET /dealer/int:dealer_id/add
- Отображение формы добавления автомобиля.


### POST /dealer/int:dealer_id/add
- Сохранение нового автомобиля в базе данных.

## Удаление автомобиля
### POST /car/int:car_id/delete

- Удаление автомобиля из базы данных.
